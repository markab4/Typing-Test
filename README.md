# Typing Test

A browser application to test typing speed and accuracy. Counts the number of typing errors made, calculates Words Per Minute (WPM), and randomizes test text from a collection of over 1600 famous quotes.

## Built With
1. JavaScript
2. HTML
3. CSS
4. Node-Static
